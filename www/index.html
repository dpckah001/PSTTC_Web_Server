Welcome to PSTTC Web Server

PSTTC (PowerShell Temporary Testing Console) is a simple web server implemented using PowerShell and the .NET Framework's HttpListener class. This server is designed for testing and educational purposes and is not intended for production use.

Features

  • Serves static files from the www directory.
  • Logs all access attempts to an access.log file.
  • Supports monitoring of file changes and automatic cache invalidation.
  • Easy to configure and extend.

Getting Started

To get started with PSTTC Web Server:

  1. Create a new directory for your project.
  2. Place your static files (e.g., HTML, CSS, JavaScript) in the www subdirectory.
  3. Run the psttc.ps1 script in PowerShell.

Configuration

The server can be configured by editing the psttc.ps1 script. You can change the port number, document root, and other settings as needed.

Examples

Here are some examples of how to modify the server configuration:

$port = 8080 # Change the port number
$documentRoot = "content" # Change the document root
$logFile = "logs/access.log" # Change the log file location
